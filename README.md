# 📥 Universal Video Downloader Bot 🚀

একটি শক্তিশালী এবং সুপার ফাস্ট টেলিগ্রাম বট যা ব্যবহার করে আপনি যেকোনো সোশ্যাল মিডিয়া প্ল্যাটফর্ম থেকে সরাসরি ভিডিও ডাউনলোড করতে পারবেন। কোনো অ্যাড বা থার্ড-পার্টি লিঙ্ক ছাড়াই সরাসরি ভিডিও ফাইলটি আপনার ইনবক্সে চলে আসবে।

### ✨ মূল ফিচারসমূহ:
- 🎬 **Direct Download:** সরাসরি ভিডিও ফাইল হিসেবে পাঠিয়ে দেয়।
- 🖼️ **Thumbnail Support:** ভিডিওর কভার ফটো বা থাম্বনেইল সহ আসে।
- 📝 **Title Integration:** ভিডিওর অরিজিনাল টাইটেল বা নাম সহ মেসেজ।
- 🌐 **1000+ Sites Support:** YouTube, FB, Instagram, Twitter, TikTok এবং আরও অনেক কিছু।
- ⚡ **Super Fast:** লো-লেজেন্সি এবং হাই-স্পিড ডাউনলোড।

### 🛠️ টেক স্ট্যাক (Tech Stack):
- **Language:** Python 🐍
- **Core Engine:** `yt-dlp`
- **Library:** `python-telegram-bot`
- **Environment:** Compatible with Termux, Linux, and Windows.
